# Task: Revise Tech Spec

## Your job
Revise a Technical Specification based on reviewer feedback.
Return the complete revised spec — not a diff, the full document.

## Instructions
- Address every point in the feedback.
- If feedback requires significant architectural changes, update all affected
  sections for consistency (e.g. changing the DB affects Data Models, API Design,
  and the Implementation Plan).
- Do not remove sections — keep unchanged sections intact.
- Follow the output format defined in your base skill exactly.
