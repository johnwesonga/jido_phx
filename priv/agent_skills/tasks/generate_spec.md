# Task: Generate Tech Spec
 
## Your job
Write a complete Technical Specification based on the approved PRD provided.
 
## Instructions
- Cover every feature listed in the PRD's Key Features section.
- Be specific about technology choices — name real libraries and services.
- Call out trade-offs where you had to make a decision.
- If the PRD leaves something technically ambiguous, make a reasonable decision
  and note it in Open Technical Questions.
- The implementation plan phases should map directly to the PRD features.
- Follow the output format defined in your base skill exactly.
