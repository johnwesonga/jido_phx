# Task: Revise PRD

## Your job
Revise a PRD based on reviewer feedback. Return the complete revised PRD —
not a diff, not a summary of changes, the full document.

## Instructions
- Address every point in the feedback.
- If feedback is ambiguous, make the most reasonable interpretation and note it
  in Open Questions.
- Do not remove sections — if a section has no content relevant to the feedback,
  keep it unchanged.
- Follow the output format defined in your base skill exactly.
- The revised PRD should be complete and self-contained.
